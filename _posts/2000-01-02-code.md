All open source-code I write is hosted on  [GitHub](https://github.com/johannesu).

### Algorithms
* [3D-label stereo](https://github.com/johannesu/stereo).
	<blockquote>
		Estimate a depth map from two or more images taken of the same scene using 3D-label stereo.
	</blockquote>
* [Multi-region segmentation](https://github.com/johannesu/multi-region).
	<blockquote>
		Segmentation supporting arbitrary inclusion and exclusion constrains for different regions.
	</blockquote>
* [Point-cloud regularization](https://github.com/johannesu/point-cloud).
	<blockquote>
		Curvature based point-cloud regularization.
	</blockquote>
* [Partial enumeration](https://github.com/johannesu/partial-enumeration).
	<blockquote>
		Higher-order non-submodular pseudo-boolean optimization using partial enumeration. Useful in some Computer Vision problems.
	</blockquote>
* [Parametric max-flow](https://github.com/johannesu/parametric-maxflow).
	<blockquote>
		Find <strong>all</strong> solutions to any two-parameter max-flow problem. Uses exact arithmetic and is thereby intractable slow for larger problems.
	</blockquote>
* [Curve extraction](https://github.com/PetterS/curve_extraction).
	<blockquote>
		Efficient shortest path in 2D and 3D able to regularize curvature and torsion.
	</blockquote>
